# STM32_adxl345
Библиотека для акселерометра adxl345 с интерфейсом I2C на микроконтроллере STM32F103.
#### V1.0
Измерение ускорения по 3 осям.
#### V2.0
Добавлено: 
+ Прерывание для обнаружения активности.
+ Настройка засекания двойного удара.
+ Свободное падение.
